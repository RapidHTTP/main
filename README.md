# RapidHTTP

RapidHTTP 是一个简单、高效的 Python HTTP 服务器。

## 特性

- 快速、高效的处理 HTTP 请求
- 支持静态文件服务
- 多线程处理请求
- 简单易用的 API

## 使用方法

1. 安装 Python 3.x （如果尚未安装）。
2. 下载或克隆 RapidHTTP 代码库。
3. 在命令行中进入 RapidHTTP 代码库的根目录。
4. 运行以下命令启动 RapidHTTP 服务器：

   ```
   python main.py
   ```

5. 访问 http://localhost:8000/ 以查看默认页面。
